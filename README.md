Tested with
- MSVC 14(2015) + visual_studio_multi, cmake generator
- Linux + cmake generator

You neeed to add remote repository `bincrafters`
```sh
$ conan remote add bincrafters "https://api.bintray.com/conan/bincrafters/public-conan"
```
